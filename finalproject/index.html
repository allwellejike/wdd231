Tech Learning Hub - Site Plan

A structured plan for a technology learning and support website

Site Name

Tech Learning Hub

This name was chosen because it represents a central platform where users can learn technology skills, explore web development tutorials, and access device repair support.

Site Purpose

The purpose of this website is to provide structured learning resources for HTML, CSS, and JavaScript, share modern technology trends, and offer practical troubleshooting guides for computers and mobile devices. It also includes a services section for tech support.

Scenarios

  • How can I start learning web development from scratch?
  • Where can I find beginner-friendly HTML, CSS, and JavaScript tutorials?
  • What are simple solutions for common phone or computer issues?
  • How can I request technical support or repair services?

Color Scheme

The following colors define the visual identity of the website:

  • #0A192F (Dark Blue) – Used for background, header, and main layout structure.
  • #64FFDA (Teal) – Used for headings, highlights, buttons, and accents.
  • #FFFFFF (White) – Used for body text to ensure readability and contrast.

Typography

  • Poppins – Used for all headings to create a modern tech-focused look.
  • Roboto – Used for body text for clarity, readability, and simplicity.
